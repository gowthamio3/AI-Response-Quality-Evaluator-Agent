# AI Response Quality Evaluator Agent

## Project Overview
The AI Response Quality Evaluator Agent evaluates AI-generated responses by comparing them with reliable reference knowledge. It uses a RAG (Retrieval-Augmented Generation) approach with FAISS vector search to retrieve the most relevant reference answer and evaluate the AI response.

## Features
- User-friendly Streamlit interface
- Reference Knowledge Base
- Dataset Chunking
- Embedding Generation
- FAISS Vector Database
- Semantic Retrieval
- AI Response Evaluation
- Overall Quality Score

## Technologies Used
- Python
- Streamlit
- Hugging Face Datasets
- Sentence Transformers
- FAISS
- Scikit-learn

## Datasets Used

- Custom AI Knowledge Base
